# ghost-human-demo.github.io/

This repository contains the project website for the paper **GHOST: Generalizing Skills By Hierarchical Learning from Human Demonstrations**.

**Project Website:** https://ghost-human-demo.github.io/

## About

GHOST is a hierarchical policy framework that enables robots to learn precise manipulation skills from robot data and generalize to novel objects and tasks through human demonstrations.

## Local Development

Simply open `index.html` in a web browser to preview the website locally.
