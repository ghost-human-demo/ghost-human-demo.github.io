# ghost-human-demo.github.io/

This repository contains the project website for the paper **GHOST: Hierarchical Sub-Goal Policies for Generalizing Robot Manipulation**.

**Project Website:** https://ghost-human-demo.github.io/

## About

GHOST is a hierarchical policy framework that enables robots to learn precise manipulation skills from robot data and generalize to novel objects and tasks through human demonstrations.

## Local Development

Start a local server from the repository root:

```
python -m http.server 8000
```

Then open http://localhost:8000 in your browser.
