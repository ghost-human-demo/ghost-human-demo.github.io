👻 GHOST
Generalizing Skills By Hierarchical Learning from Human Demonstrations
Sriram Krishna1, Ben Eisner1, Haotian Zhan1, Ying Yuan1, Haoyu Zhen2, Chuang Gan2,
Shubham Tulsiani1, David Held1 1Robotics Institute, Carnegie Mellon University     2UMass Amherst

GHOST learns robot skills from in-distribution teleoperation data and generalizes to novel tasks using human demonstrations. We train a hierarchical policy that decouples task-specific goal prediction (πhi) from embodiment-specific action execution (πlo). By training πhi on both robot and human data while keeping πlo purely on robot data, GHOST transfers learned manipulation skills to out-of-distribution tasks with only human video demonstrations.

Abstract.
We present GHOST, a framework for generalizing robot skills to new tasks through human demonstrations. Robot teleoperation data is critical for training robust policies, but is expensive to scale, while human video is abundant, but lacks executable robot actions. We use a hierarchical policy framework that decouples a policy into two parts. The high-level policy maps visual observations to task "sub-goals". These sub-goals are intermediate objectives like "grasp the handle" defined in a task-relevant but embodiment-agnostic manner, and do not require action labels. The low-level policy is conditioned on these goals to execute embodiment-specific actions and is trained purely on robot data. This setup provides two key benefits. First, even when trained purely on teleoperation data, it significantly boosts performance. Second, it enables out-of-distribution generalization through human demonstrations - allowing us to learn precise embodiment-specific skills from robot data and then generalize these learned skills across novel objects and compositions of these skills.
Approach.
GHOST decomposes the policy learning problem into two components: a high-level policy that predicts spatial sub-goals in a task-specific but embodiment-agnostic manner, and a low-level policy that learns embodiment-specific skills to achieve those goals. This factorization enables the high-level policy to be trained on both human and robot data, while the low-level policy is trained exclusively on robot demonstrations to learn precise manipulation behaviors.

Left: Parallel-jaw gripper. Right: MANO hand. (Interactive - drag to rotate, scroll to zoom)

We collect two types of demonstrations: robot teleoperation data and human demonstrations. For robot data, we collect demonstrations across multiple task variants, each demonstrating the same skill (e.g., pick-and-place), to provide in-distribution coverage. Sub-goals are extracted automatically by identifying timesteps where the gripper state changes. For human data, we collect demonstrations of the same skill in novel settings using new objects or novel applications. We track 3D hand poses using off-the-shelf hand pose estimators and manually annotate sub-goal timesteps.

To enable cross-embodiment transfer, we represent end-effector poses as a set of 4 3D keypoints that can be extracted from both robot grippers and human hands: the gripper base/palm, finger tips, and grasping center. This unified representation allows the high-level policy to be trained on both robot and human demonstrations.

2. High-Level Goal Prediction Policy.

GHOST High-level goal prediction architecture: RGB-D observations from multiple cameras are processed with a DINOv2 encoder, with patch tokens augmented by 3D coordinates. Additional context (gripper state, language embedding, embodiment name) is encoded via separate MLPs. An encoder-only transformer processes all tokens, with each patch predicting GMM parameters of a 3D goal distribution over end-effector keypoints.

The high-level policy predicts the end-effector pose at the next sub-goal timestep. We parameterize it as an encoder-only transformer operating on RGB-D observations from multiple cameras. Each RGB-D image is processed with a frozen DINOv2 encoder, producing patch tokens that are augmented with 3D coordinates derived from depth. To handle multimodality in demonstrations, we model the goal distribution as a per-patch Gaussian Mixture Model where each patch predicts mixing weights and 3D residual vectors relative to the patch center.
3. Low-Level Goal-Conditioned Policy.

End-effector heatmap generation for goal conditioning. Predicted 3D keypoints are projected to 2D coordinates, and then converted to dense distance field heatmaps that encode spatial proximity to each keypoint.

We instantiate the low-level policy as a Diffusion Policy that generates chunks of actions by denoising conditioned on observations and spatial goals. The 3D goals predicted by the high-level policy are projected to each camera's image plane and converted to dense end-effector heatmaps. These heatmaps encode the pixel distance field from each keypoint, providing a spatial representation that preserves structure while densifying the sparse goal information. This approach allows the policy to effectively condition on goals while leveraging standard image-based architectures.
Experiments and Results.
We evaluate GHOST across multiple challenging real-world manipulation tasks to demonstrate both in-distribution performance improvements and out-of-distribution generalization through human demonstrations.
